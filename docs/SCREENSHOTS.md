# Adding Screenshots

Add 4 screenshots to this folder with these exact filenames:
- screenshot-dashboard.png   (dashboard dark mode, full page)
- screenshot-pipeline.png    (pipeline kanban, dark mode)
- screenshot-clients.png     (clients list with search bar visible)
- screenshot-detail.png      (client detail page)

Suggested tool: browser screenshot (F12 → device toolbar → full page screenshot)
Recommended resolution: 1440×900 or wider.
